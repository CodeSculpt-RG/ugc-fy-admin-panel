# UGC FY Admin Console — Design System
**Version:** 1.0
**Aesthetic:** Premium Enterprise SaaS

---

### 1. Visual Identity
The UI should feel luxurious, minimal, and operationally powerful. It draws inspiration from systems like Stripe, Linear, and Vercel.

### 2. Color Palette
| Role | Hex | Usage |
| :--- | :--- | :--- |
| **Primary Blue** | `#2563EB` | Actions, Links, Active States |
| **Accent Orange** | `#F97316` | Highlights, Hover effects |
| **Black** | `#000000` | Deep backgrounds |
| **Dark Surface** | `#0F172A` | Sidebars, Cards, Modals |
| **Border Gray** | `#1E293B` | Dividers, Component borders |
| **Soft White** | `#F0F0FB` | Primary text, Subtle backgrounds |
| **Success** | `#10B981` | Positive status, Completed |
| **Error** | `#EF4444` | Alerts, Negative actions |
| **Warning** | `#F59E0B` | Pending states, Cautions |

### 3. Typography
- **Font Family:** `Inter` (Sans-serif)
- **Hierarchy:**
  - `H1`: 32px, Bold
  - `H2`: 24px, Semibold
  - `Body`: 16px, Regular
  - `Small`: 14px, Medium

### 4. Layout Architecture
- **Sidebar:** 280px fixed width, dark premium surface with glass effect.
- **Navbar:** Floating or top-fixed, minimal with breadcrumbs and search.
- **Main Area:** Centered or wide-contained, using soft white for content backgrounds.
- **Cards:** 24px rounded corners, subtle shadows, 1px borders.

### 5. Interaction & Motion
- **Library:** Framer Motion.
- **Style:** Smooth, fast, minimal.
- **Effects:** 
  - Glassmorphism on overlays.
  - Soft glow on hover for KPI cards.
  - Slide-in transitions for side panels.

### 6. Component Guidelines
- **Tables:** Sticky headers, 72px row height, 18px radius.
- **Buttons:** Subtle gradients, rounded-lg, clear focus states.
- **Badges:** Soft background colors with high-contrast text.
